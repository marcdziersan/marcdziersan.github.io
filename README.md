# Portfolio Marcus Dziersan

Persönliche Portfolio-Webseite von Marcus Dziersan, angehender Fachinformatiker für Anwendungsentwicklung aus Lünen. Die Seite stellt Profil, technische Schwerpunkte, Projektarbeiten, Timeline und Kontaktmöglichkeiten kompakt dar.

## Ziel der Seite

Das Portfolio richtet sich an Unternehmen, Teams und Entscheider, die sich einen schnellen Eindruck von praktischer Erfahrung, Arbeitsweise und technischen Projekten verschaffen möchten.

Der Fokus liegt auf:

- Webentwicklung mit HTML, CSS, JavaScript, PHP und SQL
- praxisnahen Anwendungen, Tools und Verwaltungsoberflächen
- Java-, Python- und Datenbankprojekten
- sauberer Struktur, nachvollziehbarer Dokumentation und kontinuierlicher Weiterentwicklung
- Einstiegsmöglichkeiten für betriebliche Praxis, Trainee- oder Junior-Positionen

## Projektstruktur

```text
.
├── index.html
└── assets/
    └── profile.jpg
```

## Dateien

- `index.html` – vollständige Onefile-Webseite mit HTML, CSS und Vanilla JavaScript
- `assets/profile.jpg` – Profilbild für die Hero- und Profilsektion

## Technische Merkmale

- responsive Single-Page-Portfolio
- keine externen Frameworks
- keine externen Schriftarten
- keine CDN-Abhängigkeiten
- kein Tracking
- keine Cookies durch die Seite selbst
- semantische HTML-Struktur
- mobile-first Navigation
- animierte Projekt-, Skill- und Timeline-Sektionen

## Inhaltliche Schwerpunkte

Die Seite zeigt eine kompakte Entwicklerpositionierung mit praktischer Projekterfahrung. Im Mittelpunkt stehen sichtbare Projektarbeit, technische Lernkurve, nachvollziehbare Umsetzung und die Bereitschaft, weitere betriebliche Praxis zu sammeln oder direkt in eine Junior-Rolle einzusteigen.

## Lokale Nutzung

Die Webseite kann direkt im Browser geöffnet werden:

```text
index.html
```

Ein lokaler Server ist nicht zwingend erforderlich. Für eine realistische Vorschau kann die Seite aber auch über einen einfachen lokalen Webserver gestartet werden.

Beispiel mit Python:

```bash
python -m http.server 8000
```

Danach ist die Seite lokal unter `http://localhost:8000` erreichbar.

## Veröffentlichung

Für den Livegang reicht es, `index.html` und den Ordner `assets` auf den Webspace hochzuladen. Die vorhandene Struktur sollte beibehalten werden, damit das Profilbild korrekt geladen wird.

## Datenschutz

Die Seite ist bewusst datensparsam aufgebaut. Sie verwendet keine externen Bibliotheken, kein Tracking, keine Analysewerkzeuge und keine eingebetteten Drittanbieterinhalte. Beim Betrieb auf einem Webserver können technisch notwendige Server-Logfiles durch den Hosting-Anbieter entstehen.
